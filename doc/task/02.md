## Task 2.1: Fake Door Test Pricing

**Mengapa:** Sebelum menentukan harga final, perlu data real: berapa orang mau bayar? Berapa yang milih varian murah vs mahal?

### Langkah detail:

- [ ] Tentukan 3 varian harga untuk di-test:
  - Varian A: Rp49.000/bulan (murah, impulsive)
  - Varian B: Rp79.000/bulan (tengah, paling realistis)
  - Varian C: Rp129.000/bulan (premium, value positioning)
- [ ] Buat 3 halaman pricing berbeda (bisa via A/B test atau ganti manual tiap 2-3 hari)
- [ ] Test Varian A: 2-3 hari — catat berapa klik "Pilih Paket"
- [ ] Test Varian B: 2-3 hari — catat berapa klik
- [ ] Test Varian C: 2-3 hari — catat berapa klik
- [ ] Analisis hasil: varian mana paling banyak diklik?
- [ ] Catat: ini data awal, bukan keputusan final. Keputusan final di Week 3.

**Format catatan hasil:**
```
Varian A (Rp49K): 3 hari, X clicks, Y conversion rate
Varian B (Rp79K): 3 hari, X clicks, Y conversion rate
Varian C (Rp129K): 3 hari, X clicks, Y conversion rate
Kesimpulan sementara: ...
```

## Task 2.4: Blog SEO

**Mengapa:** Blog adalah aset SEO jangka panjang. Setiap artikel = pintu masuk organik dari Google.

### Langkah detail:

- [ ] Riset keyword di Google / Ubersuggest:
  - Cari: "ide bermain anak", "aktivitas anak di rumah", "soft skill anak"
  - Catat volume pencarian + suggestions
- [ ] Tulis 5 artikel:
  - [ ] Artikel 1: **"10 Ide Bermain Anak di Rumah Tanpa Gadget"**
    - Target keyword: "ide bermain anak di rumah"
    - Struktur: intro → 10 ide (dengan deskripsi singkat) → CTA daftar early access
  - [ ] Artikel 2: **"Cara Melatih Soft Skill Anak Usia 3-6 Tahun"**
    - Target: "soft skill anak", "cara melatih soft skill anak"
    - Struktur: apa itu soft skill → kenapa penting → cara melatih → CTA
  - [ ] Artikel 3: **"Aktivitas Seru untuk Anak Hiperaktif"**
    - Target: "aktivitas anak hiperaktif", "anak hiperaktif"
    - Struktur: ciri anak hiperaktif → 8 aktivitas penyalur energi → CTA
  - [ ] Artikel 4: **"5 Tanda Anak Siap Sekolah (Bukan Calistung)"**
    - Target: "tanda anak siap sekolah", "kesiapan sekolah anak"
    - Struktur: 5 tanda (kemandirian, sosial, emosi, komunikasi, motorik) → CTA
  - [ ] Artikel 5: **"Worksheet Anak Gratis untuk Latihan Kemandirian"**
    - Target: "worksheet anak", "lembar kerja anak tk"
    - Struktur: 5 worksheet yang bisa di-download → link download (gated email) → CTA
- [ ] Di akhir setiap artikel, tambah CTA:

## Notes

- **Konten:** Kuantitas > kualitas untuk sekarang. 1 konten biasa yang naik > 1 konten sempurna yang ga pernah di-post.

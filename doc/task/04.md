## Task 4.4: Trial → Paid Monitoring

### Metrik yang dipantau:

- [ ] **Conversion rate:** trial → paid
  - Target: ≥5%. Kalau <5%, ada masalah.
  - Cek: pricing terlalu mahal? onboarding kurang jelas? value tidak terasa dalam 7 hari?
- [ ] **Retention rate:**
  - D1: user balik setelah H+1? (target: >60%)
  - D3: user balik setelah H+3? (target: >40%)
  - D7: user masih aktif di H+7? (target: >25%)
- [ ] **North Star Metric:** Aktivitas offline selesai per anak per minggu
  - Target: >5 aktivitas/minggu/user
  - Kalau <5: aktivitas tidak menarik? rekomendasi tidak tepat?

### Action jika conversion rendah:

- [ ] **Kalau conversion < 5%:**
  - Cek onboarding: apakah user paham cara pakai?
  - Cek aktivitas: apakah rekomendasi sesuai usia?
  - Cek harga: bandingkan dengan fake door test
  - Coba extend trial ke 14 hari (kalau 7 hari terlalu pendek)
- [ ] **Kalau retention rendah:**
  - Cek notifikasi: apakah user dikasih pengingat?
  - Cek konten: apakah aktivitas cukup bervariasi?
  - Cek UI: apakah user bisa dengan mudah menemukan aktivitas baru?

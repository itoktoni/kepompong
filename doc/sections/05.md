## 5. Activation

### Current state

Pre-launch. Tidak ada produk live untuk diaktifasi. Aktivasi saat ini berarti: (1) konversi pengunjung landing page menjadi signup waiting list, dan (2) konversi signup waiting list menjadi partisipan beta.

Tidak ada onboarding flow, tidak ada paywall, tidak ada first-session experience — semuanya akan dibangun bersamaan dengan MVP.

### The plan

**Move 1 — Landing page conversion (pre-launch activation)**

Landing page adalah satu-satunya surface konversi sebelum produk live. Target: visitor → signup waiting list.

Elemen landing page:
- **Headline:** "3 detik dapat ide aktivitas untuk soft skill anak — tanpa screen time."
- **Subhead:** "Kepompong bantu Bunda membentuk karakter Si Kecil lewat aktivitas offline, sesuai usia dan target skill."
- **Visual:** Mockup aplikasi (3 kartu aktivitas) atau foto ibu dan anak bermain
- **Social proof:** Jumlah waiting list signups (setelah mulai terkumpul)
- **CTA:** "Daftar Early Access Gratis" → form (nama, email, usia anak)
- **Fake door:** Tombol "Dapatkan Rekomendasi" → "Segera Hadir! Daftar waiting list dan dapatkan notifikasi pertama."

Target konversi: >5% visitor → signup (benchmark promising).

**Move 2 — Onboarding kilat (<30 detik)**

Saat MVP launch, onboarding harus selesai dalam <30 detik. Ini adalah competitive advantage — mayoritas aplikasi parenting punya onboarding bertele-tele.

Flow onboarding:
1. Layar 1: "Halo Bunda! Siapa nama Si Kecil?" (input nama)
2. Layar 2: "Berapa usia Si Kecil?" (pilih range atau input tanggal lahir)
3. Layar 3: "Hari ini mau latih soft skill apa?" (pilih 1 dari 5 pilar — Karakter, Sosial-Emosional, Kemandirian, Komunikasi, Resiliensi)
4. Layar 4: "Main di dalam atau luar rumah?" (Indoor/Outdoor)
5. → **Langsung ke halaman rekomendasi.** 3 kartu aktivitas muncul.

Tidak ada tutorial. Tidak ada "saksikan video ini." Tidak ada "isi profil lengkap dulu." Value prop harus dirasakan dalam 30 detik pertama.

**Move 3 — First recommendation experience (the "aha moment")**

Aktivasi terjadi saat orang tua melihat 3 kartu aktivitas yang relevan dengan anaknya. Momen ini harus:
- **Cepat:** <3 detik dari klik
- **Personal:** Aktivitas sesuai usia anak, target skill, dan kondisi (indoor/outdoor)
- **Konkret:** Bisa langsung dilakukan — bukan saran abstrak
- **Printable:** Tombol "Cetak Worksheet" 1 klik

Setelah rekomendasi pertama:
- CTA: "Simpan ke Jadwal" → orang tua bisa planning
- CTA: "Coba Aktivitas Lain" → eksplorasi database
- CTA: "Daftar Premium (Gratis 7 Hari)" → aktivasi trial

**Move 4 — Beta activation pipeline (pre-MVP)**

Sebelum MVP launch, aktivasi berarti mengubah signup waiting list menjadi partisipan beta yang engaged.

Pipeline:
1. Signup waiting list → auto-email: "Terima kasih, Bunda! Kepompong akan segera hadir."
2. Minggu 2: Email kedua: "Sambil menunggu, ini 5 aktivitas gratis untuk Si Kecil." (PDF worksheet)
3. Saat beta siap: Email ketiga: "Bunda terpilih sebagai early adopter! Masuk ke grup WA beta."
4. Di grup WA: kirim aktivitas harian manual → minta feedback → ukur Sean Ellis

**Move 5 — Paywall + trial flow (saat MVP launch)**

Paywall hadir setelah user merasakan value (bukan sebelum).

Flow:
1. Hari 1-7: semua fitur gratis (termasuk premium features)
2. Hari 7: notifikasi "Masa trial Bunda akan berakhir besok. Si Kecil sudah selesaikan X aktivitas minggu ini!"
3. Hari 8: paywall muncul saat user mencoba fitur premium (challenge unlimited, assessment semua dimensi)
4. Opsi: Premium Rp39rb/bln, Premium+ Rp69rb/bln, atau lanjut Gratis

Pertimbangan: harga Rp39rb = segelas bubble tea. Impulse-buy friendly.

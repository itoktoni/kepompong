## 11. Marketing Operations Stack

### The thesis

Satu founder + fCMO (agentic tooling) = output yang biasanya membutuhkan tim 5-8 orang. Skills library + MCP integrations memungkinkan founder non-teknis menjalankan SEO, konten, sosial media, email lifecycle, dan referral program tanpa dedicated hire di setiap channel.

### Skills mapped to AARRR stages

| Stage | Primary skills | Supporting skills |
|-------|---------------|-------------------|
| Acquisition | `social`, `seo-audit`, `content-strategy` | `ai-seo`, `competitors`, `analytics`, `public-relations`, `community-marketing` |
| Activation | `signup`, `onboarding`, `copywriting` | `paywalls`, `cro`, `ab-testing` |
| Retention | `emails`, `churn-prevention` | `copywriting`, `ab-testing` |
| Referral | `referrals`, `community-marketing` | `social`, `emails`, `copywriting` |
| Revenue | `pricing`, `paywalls` | `revops`, `ab-testing` |
| Cross-cutting | `product-marketing`, `marketing-ideas`, `analytics` | ‚Äî |

### MCPs / APIs mapped to stages

| Stage | Existing connections | fCMO tooling layer |
|-------|---------------------|-------------------|
| Acquisition | ‚Äî | Ahrefs API / DataForSEO (keyword riset), Typefully MCP (social scheduling), GitHub MCP (konten deploy) |
| Activation | ‚Äî | Figma (prototype), GitHub MCP (MVP build), Google Forms (beta survey) |
| Retention | ‚Äî | Customer.io / Mailchimp (email lifecycle), FCM (push notif) |
| Referral | ‚Äî | Dub.co (referral tracking), WhatsApp (komunitas) |
| Revenue | ‚Äî | Midtrans/Xendit (payment), Stripe MCP (subscription analytics) |
| Cross-cutting | ‚Äî | GA4 MCP (analytics), defuddle (riset), Notion (knowledge base) |

### Concrete example

**Pre-launch content production pipeline.** Founder menulis 1 draft artikel parenting ‚Üí submit ke `content-strategy` skill untuk review struktur SEO ‚Üí `social` skill untuk breakdown jadi 3 konten TikTok + 1 thread Threads + 1 infografis IG ‚Üí `typefully` untuk jadwal posting. Satu artikel jadi 5 aset konten tanpa tambahan kapasitas.

### Capability unlocks by funding stage

| Stage | Headcount | Tooling | Channels live |
|-------|-----------|---------|---------------|
| **Pre-seed (now)** | Founder-only + fCMO | Marketing skills + MCP layer | Organic: TikTok, IG, Threads, SEO, FB Groups, WA komunitas |
| **Seed close** | +1 lifecycle/content manager | + paid ad accounts, `ads` skill | + Paid pilot ($5-15K/mo): Apple Search Ads, Meta |
| **Series A** | + performance marketing + content | + dedicated tooling ($2-5K/mo) | + Paid scaling ($50-150K/mo), B2B vertical |

### Team and agency model (RACI)

| Function | Owned by (internal) | Executed by |
|----------|--------------------|-------------|
| Growth marketing | Founder | fCMO + agentic tooling (sekarang) ‚Üí lifecycle hire (seed) |
| Product marketing | Founder | fCMO (product-marketing skill) |
| Content marketing | Founder | fCMO (content-strategy, social skills) ‚Üí content hire (seed) |
| Paid acquisition | ‚Äî | Belum ada ‚Äî di-hold sampai seed close |
| Komunitas | Founder | Manual (sekarang) ‚Üí community manager (seed) |

## 9. 90-Day Roadmap

### Weeks 1-2 — Unblock
Highest-confidence, lowest-cost changes. Removing things that are broken (things that don't exist yet = broken).

| Move | Stage | Owner |
|------|-------|-------|
| Produksi 5 konten TikTok perdana (demo fitur, parenting tip) | Acquisition | Founder |
| Launch landing page + waiting list flow + fake door | Activation | Founder (dev) |
| Setup auto-email nurture sequence untuk waiting list (3 email) | Retention | Founder |
| Siapkan 3 varian landing page untuk fake door test pricing | Revenue | Founder (dev) |
| Design EARLYBUNDA referral program (mekanisme + reward) | Referral | Founder |
| Set up Google Search Console + GA4 untuk landing page | Cross-cutting | Founder (dev) |

### Weeks 3-4 — Foundation
Pillar/foundational work. First content. First tests live.

| Move | Stage | Owner |
|------|-------|-------|

| Join 5 grup Facebook parenting, mulai 3 kontribusi | Acquisition | Founder |
| Design onboarding flow (4 layar, <30 detik) — Figma prototype | Activation | Founder |
| Siapkan 5 worksheet PDF sebagai lead magnet | Activation | Founder |
